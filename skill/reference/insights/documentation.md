# Documentation nudge

**PM question:** Does enablement content match the promise and help users succeed?

## How to check

Find README, docs, guides, getting-started content. Compare to positioning and what customers need per memory.

## What to look for

- Intro and promise vs `PRODUCT.md` and marketing copy
- Audience tone vs ICP — too technical or too shallow?
- Getting started path to first value without hand-holding
- Docs emphasize strategy focus or document everything including non-goals?
- Stale claims vs current product or ingestion notes

Apply gap moves from `product.md`. Classify as Opportunity, Bloat, or Improvement.
