# Trial nudge

**PM question:** Can prospects try before they commit — self-serve eval, guest access, low-friction first touch?

## How to check

Simulate try-before-commit: marketing CTA → signup/auth → first value. Include technical eval if memory says dev-led (install docs, env setup, version pins, missing templates).

## What to look for

- Guest access, free tier, or trial vs gating and signup walls
- Setup friction (env templates, strict runtime requirements, missing quickstart)
- Demo-only or sales-led path when memory says self-serve try
- Evaluator walk blockers — where try motion stops cold
- Alignment with pricing and activation surfaces

Apply gap moves from `product.md`. Classify as Opportunity, Bloat, or Improvement.
