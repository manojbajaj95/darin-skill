# Activation nudge

**PM question:** Can new users reach first value quickly — signup, first run, path to aha?

## How to check

Simulate first-run from signup through first meaningful outcome. Find signup, welcome, wizard, and empty-state flows yourself.

## What to look for

- Steps and friction vs path to aha in memory
- First screen vs landing/marketing promise — bait-and-switch?
- Drop-off themes from ingestion still present in flow?
- Flow optimizes for north star vs vanity setup?
- Time-to-value vs strategy focus
